# touchpad

Script to set some properties of the touchpad, for example on startup of 
the laptop.

- enables natural scrolling
- enables tap to click
- disables 'disable tap while typing'

## Usage

Launch the script:
```
./touchpad
```

Or inside i3wm config:

```
exec PATH_TO/touchpad
```

## Requirements

- sh
- xinput
